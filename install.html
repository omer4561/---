📥 שלבי התקנה

עקוב אחרי השלבים הבאים בסדר

1

הורדת התוסף

לחץ על כפתור "הורד עכשיו" בעמוד הבית וקבל את קובץ ה-ZIP

youtube-summarizer.zip (גודל: ~500KB)

חלץ את הקובץ לתיקייה בחרת, למשל:

C:\Extensions\youtube-summarizer\
2

פתח את Chrome Extensions Page

בכרום, הקלד בשורת הכתובות:

chrome://extensions/

או בחר: ☰ (תפריט) → סכום כלים נוספים → הרחבות

3

הפעל Developer Mode

בפינה עליונה ימנית של עמוד ההרחבות:

⚙️ סמל מצב מפתח - הפוך אותו ON (כחול)

כעת אתה רואה כפתור "Load unpacked"

4

Load Unpacked

לחץ על "Load unpacked" ובחר את התיקייה שלך (שם חלצת את קובץ ה-ZIP)

✅ התוסף הוטען בהצלחה!

תראה את התוסף רשום בעמוד ההרחבות עם שם "Video Summarizer & Translator"

5

Setup Wizard (הגדרה אוטומטית)

אחרי טעינת התוסף, דף Setup יפתח אוטומטית 🎉

בדף Setup:

  • ✓ שלב 1: לחץ ע"מ להרשם ל-Claude (בחינם)
  • ✓ שלב 2: לחץ ע"מ להרשם ל-EmailJS (בחינם)
  • ✓ שלב 3: הזן את כתובת הדוא"ל שלך
  • ✓ שלב 4: בחר שפה
✅ זהו! אתה מוכן להשתמש!
6

התחל להשתמש!

1. פתח סרטון YouTube (או Udemy, Coursera, וכו')

2. לחץ על אייקון התוסף בשורת הכלים

3. לחץ "סכם ושלח"

4. חכה 30-60 שניות

5. 📧 בדוק את המייל - הסיכום הגיע!

❓ שאלות נפוצות

Q: איפה מקום הקבצים?
כשחלצת את ה-ZIP, בחר תיקייה שתזכור, למשל:
C:\Users\YourName\Downloads\youtube-summarizer
זה זה המקום ש"Load unpacked" צריך להצביע עליו.
Q: אני לא רואה את אייקון התוסף
1. בדוק שהתוסף טעון בכרום (chrome://extensions/) ✓
2. אם תוסף מוגבל - לחץ על 🧩 (puzzle) בפינה ימנית עליונה
3. חפש "Video Summarizer" ובחר PIN כדי להצמיד אותו
Q: מה זה Claude API ו-EmailJS?
Claude API: AI מ-Anthropic שמתרגם וסוכם סרטונים (בחינם! רק משלם בשימוש - ~$0.10 לסרטון)

EmailJS: שרות שליחת מיילים בחינם (200 מיילים בחודש בחינם)

Setup Wizard יפתח בשבילך את כל הקישורים להרשמה!
Q: זה בטוח?
✅ כן!
• כל הקוד בן-קוד פתוח וזמין לביקורת
• אין שרתים שלנו - הכל עובד בתוך הדפדפן שלך
• האם ההוראות שלך לא תורמות לשוקי שלישיים
Q: האם זה עובד בכל הסרטונים?
עובד בנחמד: YouTube, Udemy, Coursera, Vimeo, Teachable, Thinkific

סרטונים עם כתוביות: כן

סרטונים ללא כתוביות: יש כפתור "תמלול ידני" שמשתמש בWeb Speech API
